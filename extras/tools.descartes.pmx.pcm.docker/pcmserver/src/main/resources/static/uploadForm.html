PMX-PCM Webserver

The manual creation of architectural performance models is very complex,
time intense and error prone. The Performance Model eXtractor (PMX) tool
automates the extraction of architectural performance models form measurement
data. Currently, PMX supports logs of the Kieker Monitoring Framework
as input data format.

For more information please visit our website
